# CashFlow Bem-vindo ao **CashFlow**, uma aplicação desenvolvida em C# para auxiliar no controle financeiro pessoal ou empresarial. O CashFlow permite registrar receitas e despesas, calcular o saldo disponível e gerar relatórios financeiros detalhados, garantindo a integridade dos dados através de validações robustas. ## Funcionalidades - **Registro de Transações**: Adicione entradas (receitas) e saídas (despesas) financeiras com facilidade. - **Cálculo de Saldo**: Obtenha automaticamente o saldo disponível com base nas transações registradas. - **Relatórios Financeiros**: Gere relatórios detalhados para análise financeira e tomada de decisões informadas. - **Exportação de Despesas para Excel**: Possibilidade de exportar as despesas registradas em formato **Excel (.xlsx)**. - **Validações de Dados**: Assegure a integridade e consistência dos dados financeiros através de validações implementadas. ## Pré-requisitos Antes de iniciar, certifique-se de ter instalado: - [.NET SDK 6.0 ou superior](https://dotnet.microsoft.com/download). - SQL Server ou outro banco de dados compatível. ## Configuração do Ambiente ```sh git clone https://github.com/Carloscavalcante97/CashFlow.git cd CashFlow dotnet restore dotnet ef database update ``` ## Execução da Aplicação ```sh dotnet run --project src/CashFlow ``` A aplicação estará disponível em `https://localhost:5001` ou conforme configurado. ## Exportação de Despesas para Excel O **CashFlow** permite a exportação das despesas registradas para um **arquivo Excel (.xlsx)**, possibilitando a organização dos dados para análise financeira, auditorias e planejamento estratégico. ### Como Exportar para Excel: 1. Acesse a funcionalidade de **Relatórios** dentro da aplicação. 2. Escolha a opção de **Exportação** e selecione o formato **Excel (.xlsx)**. 3. O sistema gerará um arquivo Excel contendo todas as despesas registradas. ## Testes Para garantir a qualidade e a integridade do código, o projeto inclui uma suíte de testes.
